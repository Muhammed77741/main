### –ò—Å–ø—Ä–∞–≤–ª–µ–Ω–∏–µ 7: –ê–¥–∞–ø—Ç–∏–≤–Ω—ã–π Trailing Stop

**–§–∞–π–ª**: `live_bot_mt5_fullauto.py:46`, `live_bot_binance_fullauto.py:48`

**–ü—Ä–æ–±–ª–µ–º–∞**: –§–∏–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω—ã–π `trailing_stop_pct = 0.5` –Ω–µ —É—á–∏—Ç—ã–≤–∞–µ—Ç –≤–æ–ª–∞—Ç–∏–ª—å–Ω–æ—Å—Ç—å.

**–†–µ—à–µ–Ω–∏–µ**: –ê–¥–∞–ø—Ç–∏—Ä–æ–≤–∞—Ç—å –∫ ATR (Average True Range).

```python
def _calculate_adaptive_trailing_pct(self, entry_price, max_price):
    """–†–∞—Å—Å—á–∏—Ç–∞—Ç—å trailing % –Ω–∞ –æ—Å–Ω–æ–≤–µ ATR"""
    # –ü–æ–ª—É—á–∏—Ç—å ATR(14)
    bars = mt5.copy_rates_from_pos(self.symbol, self.timeframe, 0, 14)
    df = pd.DataFrame(bars)
    
    # ATR calculation
    high_low = df['high'] - df['low']
    high_close = abs(df['high'] - df['close'].shift())
    low_close = abs(df['low'] - df['close'].shift())
    tr = pd.concat([high_low, high_close, low_close], axis=1).max(axis=1)
    atr = tr.rolling(14).mean().iloc[-1]
    
    # –†–∞—Å—Å—Ç–æ—è–Ω–∏–µ –æ—Ç entry –¥–æ max
    profit_distance = abs(max_price - entry_price)
    
    # Trailing –¥–æ–ª–∂–µ–Ω –±—ã—Ç—å >= 1.5 * ATR –æ—Ç max price
    min_trailing_distance = 1.5 * atr
    trailing_pct = min(0.8, min_trailing_distance / profit_distance)
    
    return max(0.3, trailing_pct)  # –ù–µ –º–µ–Ω—å—à–µ 30%, –Ω–µ –±–æ–ª—å—à–µ 80%
```
